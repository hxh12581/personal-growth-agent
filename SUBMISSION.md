## 项目描述（直接复制到申请表）

### 核心痛点

Personal Growth Agent (PGA) 旨在解决个人成长过程中面临的深层心理障碍与效率问题：

1. **CPTSD（复杂性创伤后应激障碍）**：童年创伤导致的冻结反应、回避行为
2. **拖延症**：知道该做但无法启动的 paralysis 状态
3. **过度准备与完美主义**：在准备阶段无限循环，无法进入执行
4. **学习效率低下**：缺乏科学方法论指导，事倍功半
5. **目标管理混乱**：大目标无法拆解，不知如何下手

现有工具（待办事项、番茄钟）只能解决表层问题，无法触及根本原因，更无法提供从诊断到执行的完整闭环。

### 核心逻辑流程

#### 1. 长链推理 (Long-Chain Reasoning)

**7 阶段深度分析流程**：

```
用户输入问题
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 1: 现象取证                                             │
│ • 提取客观事实，过滤主观情绪                                  │
│ • 识别矛盾点和异常现象                                        │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 2: 假设生成 (MECE原则)                                  │
│ • 从心理/行为/环境/生理多维度生成假设                         │
│ • 特别关注反常识假设                                          │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 3: 证据搜集                                             │
│ • 为每个假设寻找支持/反对证据                                 │
│ • 排除不合理假设，保留候选                                    │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 4: 深度溯源 (5Why)                                      │
│ • 连续追问"为什么"直至根本原因                                │
│ • 系统动力学分析                                              │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 5: 本质建模                                             │
│ • 构建统一解释框架                                            │
│ • 一句话概括本质                                              │
│ • 生成可验证预测                                              │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 6: 方法匹配                                             │
│ • 根据本质匹配科学方法论（CBT、福格模型、刻意练习）          │
│ • 生成微课堂讲解                                              │
└──────────────────────────────────────────────────────────────┘
    ↓
┌──────────────────────────────────────────────────────────────┐
│ Phase 7: 计划生成与执行                                       │
│ • 目标层级拆解（年→月→周→日→最小行动）                      │
│ • 设计触发器和追踪机制                                        │
└──────────────────────────────────────────────────────────────┘
    ↓
输出：完整的个性化成长方案
```

#### 2. 多 Agent 协作

**5 个专业 Agent**：

| Agent | 职责 | 核心技术 |
|-------|------|----------|
| **DeepTrace** | 深度溯源分析 | 六步本质挖掘、5Why分析、毛选矛盾论 |
| **KnowledgeClassroom** | 知识课堂讲解 | 微课堂生成、方法论匹配、考频分析 |
| **MicroProgress** | 微进度加速 | 目标智能拆解、最小行动设计、触发器设计 |
| **ActionTracking** | 行动追踪 | 游戏化系统、即时反馈、进度可视化 |
| **ContentAtomization** | 内容原子化 | 日记结构化、四维度标签、知识库构建 |

**协作模式**：
- **顺序执行**: Trace → Knowledge → Progress → Action → Atomization
- **动态路由**: 根据问题类型自动选择 Agent 链
  - 心理问题: Trace → Knowledge → Progress → Action
  - 学习问题: Knowledge → Progress → Action
  - 执行问题: Progress → Action

### 核心创新点

1. **方法论融合创新**: 首次将 CBT 认知行为疗法、福格行为模型、刻意练习、毛选矛盾论与实践论系统整合到 AI Agent 中

2. **长链推理突破**: 7 阶段深度分析流程，远超常规对话式 AI 的浅层响应（通常只有 1-2 轮），实现真正的"认知考古"

3. **多 Agent 协作架构**: 5 个专业 Agent 动态协作，支持顺序/并行/自适应执行模式，覆盖从诊断到执行的全流程

4. **个性化适配能力**: 根据用户心理特征、学习风格、时间资源自动调整策略，实现千人千面的成长方案

5. **完整闭环设计**: 现象→本质→方法→行动→复盘的完整闭环，支持持续迭代优化

### 应用场景

**场景 1: CPTSD 与拖延疗愈**
```
用户: "我总是拖延重要的工作，明明知道应该做，但就是开始不了"

系统处理:
1. DeepTrace → 识别为 CPTSD 导致的"冻结"反应，根本原因是童年"表现=价值"的条件式关爱
2. Knowledge → 匹配 CBT 认知重构 + 自我同情方法
3. MicroProgress → 拆解为"2分钟启动"最小行动
4. ActionTracking → 设置每日打卡追踪
5. Atomization → 结构化记录每次突破

输出: 包含心理疗愈 + 行动突破的完整方案
```

**场景 2: 考研学习规划**
```
用户: "我想3个月掌握408考研，如何高效复习？"

系统处理:
1. Knowledge → 考频分析（数据结构45%、计组25%、OS20%、网络10%）
2. Knowledge → 匹配间隔重复、主动回忆方法论
3. MicroProgress → 智能拆解3个月目标为周/日计划
4. ActionTracking → 刷题追踪 + 游戏化成就系统
5. Atomization → 错题原子化 + 标签管理

输出: 个性化学习计划 + 进度追踪系统
```
